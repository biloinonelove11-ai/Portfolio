# Samuel Girma - Full Stack Developer & Cybersecurity Expert

![Portfolio Banner](/src/assets/360_F_419269782_9LsP3TQndMVnZ2j3ZhTPhMjaqQpFAth9.jpg)

A modern, high-performance portfolio website built with a focus on futuristic design, glassmorphism aesthetics, and seamless user experience.

## 🚀 Features

- **Futuristic Design**: A custom Glassmorphism design system with deep blue tones, neon accents, and glowing effects.
- **Responsive Layout**: Fully responsive interface that adapts perfectly to all device sizes.
- **Dynamic Animations**: Smooth transitions and interaction effects powered by Framer Motion.
- **Performance Optimized**: Built with Vite for lightning-fast HMR and production builds.
- **Interactive UI**: Rich component library based on Radix UI and Shadcn/ui.

## 🛠️ Tech Stack

- **Framework**: [React](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [Shadcn/ui](https://ui.shadcn.com/) (Radix UI)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Routing**: [React Router](https://reactrouter.com/)

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/my-portfolio.git
   cd my_portifolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:8080`.

## 🏗️ Build

To create a production build:

```bash
npm run build
```

The built files will be in the `dist` directory.

## 🎨 Design System

The project uses a custom Tailwind configuration extending the default theme with:
- **Fonts**: Space Grotesk (Body) and Orbitron (Headings).
- **Colors**: Deep space blues, electric cyan, and neon accents.
- **Effects**: Custom backdrop blurs, glows, and animations.
